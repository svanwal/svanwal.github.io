GR Analysis

The nonprofit organization Grote Routepaden manages long-distance hiking trails in Flanders and has established several quality criteria to which these trails should conform. This project uses data analysis techniques with Open Street Maps data, in order to quantify the degree to which the GR trails satisfy these quality criteria.

GR Tracker

This web app allows users to register hikes on the Belgian long-distance GR hiking network. The app provides maps that highlight which portions of the trails users have walked on. It also calculates statistics such as the completion % of each trail. Users can also view the hikes and completion percentages of their friends.
